01 ·What it does

Cluely's feature set, your control plane.

Same overlay UX as the closed-source incumbent. None of the lock-in. Every input and every output stays on your machine unless you wire it otherwise.

/ 01 Stealth

Invisible to screen capture.

Electron's setContentProtection(true) hides the overlay from screen-record and screen-share APIs on macOS and Windows. Always-on-top, click-through optional.

/ 02 STT + TTS

Local audio, your voice.

Speech-to-speech pipeline modeled on huggingface/speech-to-speech. Optional Coqui XTTS-v2 voice cloning so the AI whispers in your voice through your earpiece.

/ 03 Two providers

Ollama + OpenRouter, side by side.

Per-agent model + fallback chain. Run interview-coding on Sonnet, run notetaker on local Llama. Switch at any time. cue model list.

/ 04 Agents

Every agent is a TOML file.

System prompt, model, hotkey, KB scope. Five built-ins ship; create more with cue agent new. Hot-swap mid-call via ⌘1-5.

/ 05 Daemon

HTTP API under everything.

FastAPI service with OpenAPI docs at /docs. The Electron overlay and CLI are both clients. Build your own — Raycast, OBS, mobile.

/ 06 CLI-first

Run interviews from the terminal.

cue ask, cue practice, cue session record, cue model benchmark. Power users skip the GUI entirely.

03 ·Quickstart

Install to first answer in two minutes.

  1. Install

    pip install cue for daemon + CLI. Add cue[stt,tts,kb] later for the heavier optional engines.

  2. Configure

    cue init writes ~/.cue/config.toml. Drop in your OpenRouter key or skip and run Ollama-only.

  3. Ask

    cue ask "talk about a hard project" streams an answer through the default agent.

  4. Daemon up

    cue daemon start exposes the full pipeline at 127.0.0.1:7821. Visit /docs for OpenAPI.

  5. Overlay

    Run cue start (or cd electron && npm start) for the stealth Cluely-style overlay.

04 ·HTTP API

A daemon under everything.

The GUI calls these endpoints. The CLI calls these endpoints. Anyone can. OpenAPI lives at http://127.0.0.1:7821/docs.

MethodPathPurpose
POST/v1/chatSSE token stream through any agent + model
GET/v1/agentsList all agents (built-in + user)
POST/v1/agents/:slugCreate or update an agent
DELETE/v1/agents/:slugRemove a user agent
GET/v1/modelsBoth Ollama + OpenRouter catalogues, side by side
POST/v1/sttAudio in → transcript chunks (0.2)
POST/v1/ttsText in → audio out (0.2)
GET/v1/sessionsPast session transcripts + answers
GET/v1/healthDaemon + provider status
